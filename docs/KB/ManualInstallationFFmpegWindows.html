Manual Installation of FFmpeg on Windows¶

v2.0 and greater of the Spatial Workstation requires FFmpeg to be installed in a specific location for the video player to work. FFmpeg should be automatically downloaded and installed when you run the Spatial Workstation installer, but there are cases when this might fail.

If you face issues with the Video Player, try manually installing FFmpeg with the steps below:

  1. Download FFmpeg for Windows from here
  2. Unzip the archive into C:\ProgramData\FB360 Spatial Workstation. Note: the ProgramData folder is typically hidden on Windows. You would either need to unhide the folder or type “C:\ProgramData” in the address bar. You might also be required to create the “FB360 Spatial Workstation” folder.
