What To Include When Reporting an Issue¶

Before reporting an issue make sure that it’s not covered in Troubleshooting, the FAQ or the User Guide. Check that the question hasn’t been asked before, a quick search of relevant keywords on the Facebook group might turn up the answer you were looking for without having to wait for someone to respond. If your problem includes additional software or services (such as the Oculus Video Player) check their support information to be sure you have everything set up correctly.

If you haven’t found the answer you are looking for then write a post on the Facebook support group. Try to include the following information in your post as this will make it easier to reproduce the issue and make it faster to find a fix:

  • What the problem is
  • Steps on how to reproduce the issue
    • An example would be:
      1. Set the elevation to 30°
      2. Disable ‘Get From Listener’
      3. Set Listener Roll to 13.7°
  • OS Version
  • Mac Model
  • DAW version
