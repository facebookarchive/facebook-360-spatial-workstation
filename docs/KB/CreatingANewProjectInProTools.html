Creating A New Project In Pro Tools 10 and 11¶

As of version 3.0.0 of the Spatial Workstation Pro Tools HD 10 and 11 are no longer supported. Pro Tools HD 10 and 11 can still be used with previous versions of the Spatial Workstation, which can be downloaded here.

To create a new project with the provided template and I/O options:

Pro Tools 11 and later¶

  1. Open Pro Tools HD
  2. Create a new project
  3. Select Create Session From Template…
  4. Choose Post-Production from the drop-down menu
  5. Select either FB360 2nd Order Ambisonics or FB360 3rd Order Ambisonics depending on your project requirements
  6. Set the sample rate and bit depth of your session (see Sample Rate Considerations for information about target sample rates)
  7. Click Ok

Pro Tools 10¶

  1. Open Pro Tools HD
  2. Go to File ‣ Open Project and open either FB360 2nd Order Ambisonics.ptxt or FB360 3rd Order Ambisonics.ptxt depending on your project requirements. These templates are found in /Users/UserName/Documents/Pro Tools/Session Templates/Post Production/FB360 3D Audio.ptxt on macOS or C:\UsersPublic\Documents\FB360 Spatial Workstation\Pro Tools\Session Templates\Post Production\ on Windows
  3. Choose your session sample rate and bit depth
  4. Pro Tools will now prompt you to save the template as a new session
  5. Save your project
